# nodeTemplate
## A simple nodejs template to get you started.

## The template is a great start for nodejs backend development

Includes:
* hbs as template/view engine
* bootstrap 4.5
* Some node modules to avoid the hassle of starting from scratch
* FontAwesome
* Blank file for db config

## Usage:
* clone the repository: ```git clone https://github.com/Amaanx2056x/nodeTemplate.git  my-app```
* Install the dependencies: ```npm install```
* Start the app: ```npm start```
